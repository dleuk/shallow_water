Core module defining the SWE state vector and the right-hand-side (RHS) of the 2-D shallow water equations.

The 2-D SWE in conservation form are:

∂h/∂t + ∂(hu)/∂x + ∂(hv)/∂y = 0 ∂(hu)/∂t + ∂(hu²+g h²/2)/∂x + ∂(huv)/∂y = -g h ∂b/∂x ∂(hv)/∂t + ∂(huv)/∂x + ∂(hv²+g h²/2)/∂y = -g h ∂b/∂y

where h = water depth, u = x-velocity, v = y-velocity, b = bed elevation, g = gravitational acceleration.
