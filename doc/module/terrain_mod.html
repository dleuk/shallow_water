Bathymetry / terrain module.

The bottom topography b(x,y) enters the SWE source term. This module defines the data structure and provides procedures to initialise b on a CartesianGrid for a selection of analytic shapes (flat bottom, Gaussian hill, step, …).


Uses

    • constants_mod
